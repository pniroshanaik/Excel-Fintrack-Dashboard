# 📊 Excel Fintrack Dashboard

## 🔍 Overview

This project showcases an interactive financial dashboard built in Microsoft Excel to analyze income, expenses, and savings patterns. It enables users to make data-driven financial decisions using intuitive and dynamic visualizations.

## 🖼 Dashboard Preview

![Dashboard Preview](images/dashboard-preview.png)

## 🚀 Features

* Dynamic KPI tracking (Income, Expenses, Savings)
* Interactive filters and slicers for customized analysis
* Visual insights using charts and graphs
* Monthly financial trend analysis
* Category-wise expense breakdown

## 🛠 Tools & Techniques

* Microsoft Excel
* Pivot Tables & Pivot Charts
* Data Cleaning & Transformation
* Data Visualization

## 📂 Project Structure

* `dashboard/` → Final Excel dashboard
* `images/` → Dashboard screenshots
* `data/` → Raw dataset (if applicable)

## 📈 Key Insights

* Identified high-spending categories impacting savings
* Analyzed monthly income vs expense trends
* Highlighted opportunities for better financial planning

## 🎯 Business Impact

This dashboard helps users monitor financial health, optimize spending, and improve savings decisions using data-driven insights.
